キーボードマクロ設定

    新しいホットキーを追加

    よく使うキー:

    キーボードショートカットヘルプ

    修飾キー

    • Ctrl - コントロールキー
    • Alt - オルトキー
    • Shift - シフトキー

    特殊キー

    • F1 - F12 - ファンクションキー
    • ↑ - 上矢印キー
    • ↓ - 下矢印キー
    • ← - 左矢印キー
    • → - 右矢印キー
    • Enter - エンターキー

    よく使う組み合わせ

    • Ctrl + C - コピー
    • Ctrl + V - 貼り付け
    • Ctrl + X - 切り取り
    • Ctrl + A - すべて選択

    操作方法

    • キー入力フィールドをクリックして、実際のキーを押してください
    • 修飾キー（Ctrl、Alt、Shift）は他のキーと組み合わせて使用できます
    • テンプレートボタンをクリックして、よく使う組み合わせを素早く入力できます
